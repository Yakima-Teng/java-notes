# 访问修饰符

访问修饰符——可以修饰属性和方法的访问范围。

| 访问修饰符    | 本类        |同包       | 子孙类    | 其他        |
|:------------- |:-----------:|:---------:|:---------:|:-----------:|
| public        | √           | √         |        √  | √           |
| protected     | √           | √         |        √  | ×           |
| friendly(默认)| √           | √         |        ×  | ×           |
| private       | √           | ×         |        ×  | ×           |
